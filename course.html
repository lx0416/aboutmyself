Fake News Detection


Technologies:
Python Tensorflow





This project originated from the problem of fake news being ever more prevalent, regardless of geographical location. The dataset was acquired via Kaggle . The data was sufficiently cleaned using methods such as tokenization and lemmatization.

Text representation methods such as Bag of Words (BoW) and TF-IDF were applied to generate a baseline model and a classification approach respectively. Baseline model utilises Multinomial Naive Bayes, which generated a 94% accuracy, while the classification approach utilises Random Forest, with a 98% accuracy.

Multilingual Automatic Speech Recognition (ASR) System


Technologies:
Python, Mozilla DeepSpeech, Google Speech Recognition, Vosk






This project arose to be integrated as an airport virtual assistant. It will be installed in an environment that can be extremely noisy, so, the ASR system is configured to handle this situation by implementing a moving average low-pass filter to handle noisy environments.

The project incorporates ASR software packages in the application to avoid slow-down or interruptions to the system in the event of issues with the internet connection. It utilizes offline ASR such as Mozilla DeepSpeech, Google Speech Recognition and Vosk.

User selects any of three languages and audio files are provided to evaluate the application. The performance of using the three different software are then compared by their Word Error Rate (WER).

The repository for this website can be seen in my github profile here .

League of Legends Research Topic


Technologies:
Python



A project that researches on Champion, Damage, and Equipment in the League of Legends Esports Scene. The research questions are as follows:

  • Which champion deals the most attack damage?
  • Which champion deals the most magic damage?
  • Which champion is relatively easy to master?
  • What type of equipment is suitable for some champions?




    • The project was completed in a Jupyter notebook and the dataset was acquired via Kaggle . The data was sufficiently cleaned and analysed to answer the research questions above. An interesting insight is that there are more champions that are difficult to master, which makes LoL a relatively difficult game for new moba players. Another interesting point to note is that surprisingly, the champion with the highest damage is a tank/fighter, rather than a mage/marksman.
